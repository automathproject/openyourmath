\texte{
Dessiner la surface ${\cal S}$ d'équations paramétriques :
$\begin{cases} x = a\cos u/\ch v\cr
         y = a\sin u/\ch v\cr
         z = a(v-\tanh v)\cr \end{cases}$
où $a$ est un réel strictement positif (pseudo-sphère).
}
