\texte{
Soit $A=(0,1,0)$ et ${\cal S}$ la surface constituée des cercles verticaux de
diamètre $[A,B]$ où $B$ est un point variable sur $Ox$. Chercher une
équation cartésienne de ${\cal S}$.
}
\reponse{
$y(x^2+(y-1)^2 + z^2) = z^2$.
}
