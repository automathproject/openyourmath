\texte{
Soit $X, Y$ deux espaces topologiques séparés et $(K_n)$ une suite décroissante
de com\-pacts non vides de $X$. 
Soit $f:X\to Y$ une application continue. Montrer
que $f(\cap_n K_n)=\cap_n f(K_n)$.
}
}
