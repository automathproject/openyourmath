\texte{
Soit $K$ un convexe compact symétrique de $\Rr^n$ contenant $0$ comme point
intérieur. Alors
$K$ est la boule unité fermée associée à une norme de $\Rr^n$ : considérer pour
cela $$p(x)= \inf\{t>0 /{x\over t}\in K\}$$
}
}
