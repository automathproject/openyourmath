\contenu{
\texte{
On considère dans $M_n(\Rr)$ le sous-ensemble des matrices de déterminant
égal à $1$. Est-il compact ? On note $O(n)$ le sous-ensemble des matrices
orthogonales ($^tA.A=I$); montrer que $O(n)$ est compact.
}
}
