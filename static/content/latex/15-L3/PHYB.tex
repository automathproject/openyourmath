\contenu{
\texte{
Soit dans un espace métrique $(X,d)$ une suite $(x_n)$ telle que les trois
sous-suites $(x_{2n})$, $(x_{2n+1})$, et $(x_{3n})$ convergent. Montrer que la
suite elle-même converge.
}
}
