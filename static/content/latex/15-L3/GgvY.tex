\texte{
\label{gijsexo5}
Soit $(X,d)$ un espace métrique  compact, et
soit $f:X  \to X$ une application continue verifiant
$\forall x\neq y\ :\ d(f(x),f(y)) < d(x,y)$. Montrer qu'il
existe un point fixe unique pour $f$. (indication~:
regarder $\inf d(x,f(x))$\,)
}
}
