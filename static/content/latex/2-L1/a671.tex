\texte{
Soit $n\in \Nn^{*},x\neq 0,$ on d\'{e}finit :
$$u_{n}(x)=\left( \frac{1^{x}+2^{x}+...+n^{x}}{n}\right) ^{\frac{1}{x}}.$$

D\'{e}terminer $\ell_{n}=\lim\limits_{x\rightarrow 0}u_{n}(x).$
}
}
