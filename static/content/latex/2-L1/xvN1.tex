\texte{
Soit $(u_n)$ une suite de complexes telle que $\frac{u_1+\dots+u_n}n\to\ell\in\C$ lorsque $n\to\infty$.
Montrer que $\frac{1}{\ln(n)}\Bigl(\frac{u_1}1+\dots+\frac{u_n}n\Bigr)\to \ell$ lorsque $n\to\infty$.
}

Transformaton d'Abel.
