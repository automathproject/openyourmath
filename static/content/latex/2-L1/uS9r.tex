\texte{
Montrer que $\forall n\in \Nn^{*},\exists !x_{n}\in [n,n+1[$ solution de $
x-E(x)=\frac{1}{x^{2}}.$
Donner un \'{e}quivalent de $x_{n}$ puis faire un d\'{e}veloppement
asymptotique de $x_{n}-n$ \`{a} l'ordre 5 en fonction de $\frac{1}{n}.$
}
}
