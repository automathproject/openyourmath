\contenu{
\texte{
Soient $(u_n)$ et $(v_n)$ deux suites v{\'e}rifiant :
$\begin{cases} 0 \le u_n \le 1 \cr 0 \le v_n \le 1 \cr u_nv_n \xrightarrow[n\to\infty]{} 1.\cr\end{cases}$
Que pouvez-vous dire de ces suites ?
}
}
