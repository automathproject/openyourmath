\texte{
Soient $n\in \Nn^{*}$ et $a_{1}\leq a_{2}\leq ...\leq a_{n}$, $n$ nombres r\'{e}els.
Calculer :
$$\inf\limits_{x\in \Rr}\sum\limits_{k=1}^{n}\left| x-a_{i}\right| .$$
}
}
