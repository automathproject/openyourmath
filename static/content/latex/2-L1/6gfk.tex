\texte{
Soit $(u_{n})_{n\in \Nn}$ une suite born\'{e}e telle que :
$$\lim\limits_{n\rightarrow \infty }(u_{n}+\frac{u_{2n}}{2})=1.$$
Que dire de $(u_{n})_{n\in \Nn}$?
}
