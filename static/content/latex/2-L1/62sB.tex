\contenu{
\texte{
Soit $(u_n)$ une suite de r{\'e}els strictement positifs convergeant vers 0.
Montrer qu'il existe une bijection $\sigma : {\N} \to {\N}$ telle que
la suite $(u_{\sigma(n)})$ converge vers $0$ en d{\'e}croissant.
}
}
