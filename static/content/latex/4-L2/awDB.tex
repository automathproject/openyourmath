\contenu{
\texte{
Soit $A$ un anneau commutatif et $a,b \in A$. On dit que :
$\begin{cases} a \text{ divise } b \text{ si } b \in aA\cr
         a \text{ est premier {\`a} } b \text{ si } aA + bA =
         A.\end{cases}$

Montrer que si $a$ est premier {\`a} $b$ et $a$ divise $bc$, alors $a$ divise $c$.
}
}
