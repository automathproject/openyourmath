\contenu{
\texte{
Soit $ (A,  +, \times)$ un anneau.\\ On appelle centre de $A$ l'ensemble
$C = \left\{ x \in A / \forall y \in A, xy = yx\right\}$. \\
Montrer que $C$ est un sous-anneau de $A$.
}
}
