\texte{
Pour tout $x,y \in \R$, on pose $x
\star y = \root{\scriptstyle 3}\of {x^3+y^3}$.  Montrer que
$(\R,\star)$ est un groupe isomorphe {\`a} $(\R,+)$.
}
}
