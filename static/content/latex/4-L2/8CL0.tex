\texte{
Soit $A = (a_{i, j}) \in O_{n} (\Rr)$. Montrer que $\forall (i, j) \, \left|a_{i, j}\right|
\leq 1$ et que $\left|\sum_{i, j}a_{i, j}\right| \leq n$.
}
}
