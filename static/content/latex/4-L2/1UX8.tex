}

    \item \question{Effectuer la division euclidienne de $P$ par $x-1$.}
    \item \question{Présenter les résultats selon la méthode de Hörner.}
    \item \question{\'Ecrire $P$ dans la base $1,(x-1),(x-1)^2,\ldots, (x-1)^5$.
