\contenu{
\texte{
Soient $a,$ $b$ deux r\'eels distincts. 
Calculer le d\'eterminant suivant.
$$D_1=\left\vert\begin{array}{ccccc}a&b&\cdots&b&b\\b&a&\cdots&b&b\\
\vdots&&&&\vdots\\b&b&\cdots&a&b\\b&b&\cdots&b&a\end{array}\right\vert$$
}
}
