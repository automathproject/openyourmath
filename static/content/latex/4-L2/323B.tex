\texte{
Les nombres 119, 153 et 289 sont tous divisibles par 17. Montrer, sans
  le développer que le déterminant $\left|
   \begin{matrix}
     1&1&9\\1&5&3\\2&8&9
   \end{matrix}
  \right| $ est divisible par 17.
}
