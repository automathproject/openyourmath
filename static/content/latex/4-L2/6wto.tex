\texte{
R\'esoudre suivant les valeurs de $a \text{ et } \mu \in \Rr$
$ \begin{cases} ax + y + z + t = 1 \\ x + ay + z + t = \mu \\ x + y + az + t =
\mu^2 \\ x + y + z + at = \mu^3
\end{cases}$.
}
}
