\texte{
Montrer que toute permutation de ${\cal S}_n$ s'{\'e}crit de mani{\`e}re unique :
$\sigma = c_2^{\alpha_2}\circ c_3^{\alpha_3} \circ \dots \circ c_n^{\alpha_n}$
o{\`u} $c_i = (1\ 2\ \cdots\ i)$ et $0 \le \alpha_i < i$.
}
}
