\contenu{
\texte{
On d\'efinit sur $\Rr$ les deux lois $\oplus$ et $\otimes$ par
$x \oplus y = x + y-1$ et $x \otimes y = x + y-xy$. Montrer que
$ (\Rr, \oplus, \otimes)$ est un corps.
}
}
