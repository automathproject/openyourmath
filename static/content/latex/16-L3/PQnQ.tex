\contenu{
\texte{
Montrer que si $f(z)$ est holomorphe dans un ouvert connexe 
$\Omega $ et si $f'(z)\ne 0$ en tout point de $\Omega $, alors la 
transformation $w=f(z)$ conserve les angles.
}
}
