\texte{
Soit $c$ un nombre complexe vérifiant $|c|>e$. Montrer que
l'équation $e^z=cz$ a une solution et une seule dans $H = \{\,z\in 
\Cc\mid \Re\!e z<1\,\}$.
(On pourra considérer $D_r = H \cap D(1,r)$ avec $r\ge2$ et les fonctions
$e^z-cz$ et $-cz$.)
}
}
