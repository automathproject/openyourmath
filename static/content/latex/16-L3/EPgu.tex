\contenu{
\texte{
Montrer que si la série $\sum a_nz^n$ a un rayon $0<R<+\infty$, la série
lacunaire $\sum a_nz^{\lambda(n)}$, où $\lim{\lambda(n)\over n}=+\infty$, a
comme rayon $R'=1$. Montrer sur un exemple que la réciproque peut être fausse.
}
}
