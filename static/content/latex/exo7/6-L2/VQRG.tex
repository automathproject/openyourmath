\contenu{
\texte{
Dans un \emph{rond} on donne les droites
$D  : \begin{cases}x +2y - z = 1  \cr 2x - y +2z = 2 \cr\end{cases}$ et
$D' : \begin{cases}x + y + z = 3  \cr  x - y +2z = 0.\cr\end{cases}$

Calculer $d(D,D')$.
}

$H:(-2/19, 28/19, 35/19)$, $K:(-6/19, 40/19, 23/19)$, $d=4\sqrt{19}$.
}
