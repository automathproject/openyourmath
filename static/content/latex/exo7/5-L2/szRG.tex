\texte{
Soit $f : {[0,1]} \to \R$ continue.
Chercher $\lim_{x\to0^+}  \int_{t=0}^1 \frac{xf(t)}{x^2+t^2}\,d t$.
}
\reponse{
$\frac\pi2f(0)$.
}
