\contenu{
\texte{
Soient $a,b>0$. Calculer l'aire de l'ellipse
$E=\{ \frac{x^2}{a^2} + \frac{y^2}{b^2} \le 1 \}$ par deux m\'ethodes
diff\'erentes.

(On rappelle que l'aire d'un domaine $D$ vaut $\iint_D dx\, dy$.)
}
}
