\contenu{
\texte{
Soit $A = (a_{ij}) \in \mathcal{M}_n(\R)$.
Démontrer que : $|\det A| \le \prod_{i=1}^n \sum_{j=1}^n |a_{ij}|$.


Quand y a-t-il égalité ?
}
\reponse{
Développer le produit.

Un seul coeff. non nul par ligne et colonne, ou une ligne nulle.
}
}
