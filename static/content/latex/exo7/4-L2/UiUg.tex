\contenu{
\texte{
Soit $(P_n)_{n\in \N}$ une suite de polynômes de degrés étagés ($\deg P_n = n$).
Montrer qu'il existe un unique produit scalaire sur $\R[X]$ pour lequel la
famille $P_n$ est orthonormée.
}
