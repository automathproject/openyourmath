Soient $A$ et $B$ deux anneaux. On d\'efinit sur $A \times B$ les lois
$$ (x, y) + (x', y') = (x + x', y + y')$$
$$ (x, y) (x', y') = (xx', yy')$$
}
\begin{enumerate}
    \item \question{Montrer que $A \times B$ est alors un anneau.}

\end{enumerate}
