\chapitre{Forme bilinéaire}

\contenu{
\texte{
Soit $g$ la forme bilinéaire symétrique sur $(\Zz/2\Zz)^3$ dont la
matrice dans la base canonique est 
$\left(
 \begin{array}{ccc}
1&0&0\\ 0&0&1\\0&1&0
 \end{array}
\right)$.
Montrer que $g$ est non dégénérée. Peut-on compléter $e_1$ en une base orthogonale ?
}
}
