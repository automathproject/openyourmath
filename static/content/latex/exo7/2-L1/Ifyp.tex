\texte{
Soit $z\in\C$ tel que $|z| < 1$.
Montrer que $\sum_{n=1}^\infty \frac{z^n}{\strut 1-z^n} = \sum_{n=1}^\infty d(n)z^n$
où $d(n)$ est le nombre de diviseurs positifs de $n$.
}
}
