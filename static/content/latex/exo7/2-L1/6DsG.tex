\texte{
\'Etudier la suite $(u_{n})_{n\in \Nn}$ d\'{e}finie par :
$$u_{0}\in ]0,\frac{\pi }{2}[,u_{n+1}=\sin u_{n}.$$
Donner $\lim_{x\rightarrow 0}\frac{1}{\sin ^{2}x}-\frac{1}{x^{2}}$
,(r\'{e}ponse: $\frac{1}{3}$) en d\'{e}duire un \'{e}quivalent de $u_{n}^{-2} $
donc de $u_{n}.$
}
}
