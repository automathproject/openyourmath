\contenu{
\texte{
Trouver les fonctions $f:\R \to \R$ de classe $\mathcal{C}^2$ telles que :
$\forall\ x \in R,\ f''(x) + f(-x) = x\cos x$.
}
\reponse{
Dériver deux fois. $f(x) = \frac{\sin x-x\cos x}2 + \lambda\sh x + \mu\cos x$.
}
