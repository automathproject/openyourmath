\texte{
D\'eterminer le centre de gravit\'e du culbuto (homog\`ene), \emph{i.e.} le cône
$$\left\{ (x, y, z)\in \Rr^{3}/z \in [0, 1], x^{2} + y^{2} \leq z^{2}\right\}$$
auquel on adjoint sur sa base une demi-boule.
}
}
