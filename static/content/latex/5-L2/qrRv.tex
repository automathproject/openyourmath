\texte{
On considère les courbes planes : ${\cal Q}_i$ : $(x^2 = 2q_iy)$ et
				      ${\cal P}_i$ : $(y^2 = 2p_ix)$.
    On suppose $0 < q_1 < q_2$ et $0 < p_1 < p_2$. Calculer l'aire du
    ``quadrilatère'' limité par ${\cal P}_1$,${\cal P}_2$,${\cal Q}_1$ et
    ${\cal Q}_2$.
}
\reponse{
Formule de Green. $A = \frac43(p_2-p_1)(q_2-q_1)$.
}
