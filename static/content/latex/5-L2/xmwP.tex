\texte{
Pour $n \in \N$ on pose
$E_n = \{ (p,q)\in \N^2 \text{ tel que } p^2+q^2 \le n \}$ et $C_n = \mathrm{Card}\,(E_n)$.

Interpréter $C_n$ comme une aire et donner un équivalent de $C_n$ lorsque
$n\to\infty$.
}
}
