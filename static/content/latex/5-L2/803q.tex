\texte{
Soit $f(z) = \sum_{n=0}^\infty a_nz^n$ une série de rayon $R>0$.
Montrer, pour~$0\le r< R$~:
$\sum_{n=0}^\infty |a_n|^2r^{2n} =
\frac 1{2\pi}  \int_{\theta=0}^{2\pi}|f(re^{i\theta})|^2\, d\theta$.
}
}
