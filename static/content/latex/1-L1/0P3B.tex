\texte{
Soient $F_1$, $F_2$, $F_3$ trois sev de $E$.
Montrer que $F_1+F_2+F_3$ est directe si et seulement si :
$F_1 \cap F_2 = \{\vec 0\}$ et $(F_1 + F_2) \cap F_3 = \{\vec 0\}$.

Généraliser.
}
}
