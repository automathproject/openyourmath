\texte{
Soit $ (a, b, c, d) \in \Rr^4$ tel que $ad-bc = 1$ et $c \neq 0$. Montrer que
si $z \neq -\dfrac dc$ alors $\Im (\dfrac{az + b}{cz + d}) = \dfrac{\Im (z)}
{ \left|(cz + d)\right|^2}$.
}
}
