\texte{
Soient $E$ un ensemble et $A, B, C$ trois parties de $E$.\\
Montrer que $ (A \cup B) \cap (B \cup C) \cap (C \cup A) =
(A \cap B) \cup (B \cap C) \cup (C \cap A)$.
}
}
