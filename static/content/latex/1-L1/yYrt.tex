Montrer par r\'ecurrence sur $n$ que si $A_1, \ldots, A_n \subset E$ alors
$\mathrm{Card} (A_1 \cup \ldots \cup A_n) = \sum\limits_{k = 1}^n (-1)^{k + 1}
\sum\limits_{1 \leq i_1 < \ldots < i_k \leq n}\mathrm{Card} (A_{i_1} \cap \ldots \cap A_{i_k})$.
