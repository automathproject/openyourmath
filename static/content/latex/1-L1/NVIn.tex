\texte{
Soient $E = \left\{ (x, y, z, t) \in \Rr^4 / x + y + z + t = 0\right\}$
et $F = \left\{ (x, y, z, t) \in \Rr^4 / x + y  =  z + t \right\}$.
D\'eterminer $\dim E, \dim F, \dim (E + F), \dim (E \cap F)$.
}
