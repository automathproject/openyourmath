\texte{
Pour $k=2,3,4$ montrer que $V_{k}$ est un s.e.v. de ${ \Cc}^{k}$, et en donner
une base :
$$V_{2}=\{(a,b)\in { \Cc}^{2}/a+ib=0\},   V_{3}=\{(a,b,c)\in { \Cc}^{3} /a+2b+3c=0\},$$
$$V_{4}=\{(a,b,c,d)\in { \Cc}^{4} /a+ib=b+ic=c+id\}.$$
}
}
