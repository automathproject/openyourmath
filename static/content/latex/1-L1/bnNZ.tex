\texte{
Soient  les vecteurs $e_1=(1,2,3,4)$, $e_2=(1,-2,3,-4)$ de $\mathbb{R}^4$.
Peut-on déterminer $x$ et $y$ pour que $(x,1,y,1) \in\mbox{Vect}\{e_1,e_2\}$~?
pour que $(x,1,1,y)\in \mbox{Vect}\{e_1,e_2\}$~?
}
}
