\texte{
Soient $E$ un ev de dimension finie et $f,g \in \mathcal{L}(E)$ tels que
$\mathrm{Ker} f + \mathrm{Ker} g = \Im f + \Im g = E$.

Montrer que les sommes sont directes.
}
}
