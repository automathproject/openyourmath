\texte{
Soit $(x_n)$ une suite de r{\'e}els. On pose $y_n = \sum_{k=0}^n C_n^kx_k$.
Montrer que $(-1)^nx_n = \sum_{k=0}^n \,(-1)^kC_n^ky_k$.
}
