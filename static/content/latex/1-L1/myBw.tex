\texte{
On consid\`ere dans $\Rr^3$,  $\Pi = \text{vect}\left\{ (1, 1, 1), (1, 1, -1)\right\}$
et $D = \text{vect} \left\{ (0, 1, -1)\right\}$. Montrer que $\Rr^3 = \Pi \oplus D$.
}
