\texte{
Pour $M = \begin{pmatrix} a&b\cr c&d \cr\end{pmatrix} \in GL_2(\R)$, on note
${f_M} : {\R \cup \{\infty\}} \to {\R \cup \{\infty\}},
 x \mapsto  {\frac {ax+b}{cx+d}}$

Montrer que $M  \mapsto f_M$ est un morphisme de groupes. Quel est son noyau ?
}
