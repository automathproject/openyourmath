\texte{
Soient $x_1,x_2,x_3$ les racines de $X^3 + aX^2 + bX + c$.

Calculer le polyn{\^o}me unitaire de $\R_3[X]$ dont
$x_1^2,x_2^2,x_3^2$ sont {\it les\/} racines.
}
\reponse{
$X^3 + (2b-a^2)X^2 + (b^2-2ac)X -c^2$.
