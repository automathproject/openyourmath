\texte{
Pour $A \subset \R$ non vide et born{\'e}e, et $x \in \R$, on note :
$$d(x,A) = \inf\{ |x-a| \text{ tq } a \in A\}  \qquad (\text{\it distance de $x$ {\`a} $A$}).$$

Montrer que $\bigl|d(x,A) - d(y,A)\bigr| \le |x-y|$.
}
